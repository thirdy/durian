Durian Help

Commands

reload
Reloads overlay_config.ini and the GUI.
searchexit or sexit
Stops QIC. You'll need to run qic.ahk again to execute commands.
0,1,2..#
Generate WTB message for item # and paste to chat (You have to send it yourself).
swtb#
Saves WTB message # to savedWTB_messages.txt.
who#
Sends /whois for seller of item #.
sort*
Sort current results (see terms/sort.txt).
view#
View all stats for item #.
page#
Jump to search result page #
listleagues
Displays a List of all available Leagues you can search in.
setleague#
Writes league # to config file.
setps#
Writes PageSize # to config file.

Shortcuts

ctrl+q
Toggle the GUI ON/OFF.
PageUp / PageDown
Browse through search results.
